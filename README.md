#Markdown_cpp

Markdown converter (HTML for now) - c++ language

##Compilation

*	cd src/ && make

##How to use it?

*	./markdown-cpp
*	Enter the file
*	See the output file, in the current directory

##Done

* Bold
* Headers
* Italic
* Non-ordinate list
* Pictures
* URL
